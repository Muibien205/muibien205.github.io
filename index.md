## ⚠️ Troubleshooting common errors

Installation failures often occur for specific reasons. Check these items if the screen remains black or the app refuses to launch.

### Failed installation
If the installer stops mid-process, check your antivirus settings. Some security software detects the screen mirroring drivers as suspicious. Disable your antivirus temporarily, run the install again, and re-enable it after the process finishes.

### Driver conflicts
Windows 11 handles display drivers differently than older versions. If the install finishes but the application fails to mirror your screen, uninstall the current audio and video drivers. Reboot your machine and allow Windows to reinstall the stock drivers before you launch AirParrot 3.

### Network visibility
AirParrot 3 requires your firewall to allow connections to external receivers. When you run the software for the first time, a Windows Security Alert box will appear. Select both "Private" and "Public" network checkboxes. Click "Allow Access" to permit the application to communicate with your mirroring hardware.

### Missing prerequisites
If the software reports a missing file error, you likely need the .NET framework. Navigate to the Microsoft support website and download the latest version of the .NET Desktop Runtime. Restart your computer after installing this component and attempt the AirParrot 3 setup again.
